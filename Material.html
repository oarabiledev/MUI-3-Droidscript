Material

To use Material, you must first load the plugin at the top of your script using the LoadPlugin method like this:

 app.LoadPlugin( "Material" )

Then you can create an instance of the plugin object when you need it like this:

 plg = app.CreateMaterial()

Examples:

Example - Test

app.LoadPlugin( "Material" )

function OnStart()
{
  lay = app.CreateLayout( "Linear", "VCenter,FillXY" )

  btn = app.AddButton( lay, "Press Me" )
  btn.SetOnTouch( CallPlugin )
  plg = app.CreateMaterial()

  app.AddLayout( lay )
}

function CallPlugin()
{
  alert( plg.Test("World") )
}
